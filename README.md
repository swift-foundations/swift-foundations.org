# swift-foundations.org

Placeholder for the future [swift-foundations.org](https://swift-foundations.org) website — the public home for the Foundations layer (Layer 3) of the Swift Institute ecosystem. The Foundations layer answers *"what can be composed safely?"* — composed building blocks such as file I/O, JSON, and TLS.

Website planned for future. Current ecosystem content lives at https://swift-institute.org.
